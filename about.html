About this project

A small, opinionated starter that focuses on clarity, semantics, and responsiveness without extra tooling.

Why this exists

Many modern front‑end stacks are powerful but heavy. This project intentionally keeps things small: three HTML pages, one CSS file, and a single JavaScript file for behavior that feels native and fast in any browser.

  • Quick to understand for new developers.
  • Easy to deploy on any static hosting provider.
  • Flexible enough to adapt to different use cases.

Technical snapshot

Built with web standards only

The design language borrows from Ant Design—rounded cards, familiar blue accents, and soft shadows—implemented manually in CSS so you keep full control over the final look and feel.

Semantic HTML Flex & Grid Sticky header Mobile nav

Page structure
